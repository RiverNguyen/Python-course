# Học Python cơ bản
## 1. Biến (Variable)
### 1.1. Khai báo biến
```python
a = 5 # Integer
b = 3.14 # Float
c = "Hello" # String
d = True # Boolean
```
### 1.2. In biến
```python
print(a)
print(b)
print(c)
print(d)
```

## 2. Chuyển đổi kiểu dữ liệu (Type casting)
```python
int()
float()
str()
bool()
```

## 3. Nhập dữ liệu từ bàn phím (Input)
```python
name = input("Nhập tên của bạn: ")
print("Xin chào", name)
```
